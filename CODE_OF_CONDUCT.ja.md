

# LY Corporation's Open Source Code of Conduct_ja

本Code of Conductは、参加者に期待される行動基準を示し、包摂的で安全かつ生産的なオープンソースコミュニティを育むことを目的とするものです。

## 行動の基本原則

### 敬意を払うこと

敬意はコミュニティの基盤です。すべての参加者は、他者の多様な背景、経験、スキルレベル、意見を尊重し、意見の相違に対しては寛容と理解をもって向き合うべきです。

### 建設的にコミュニケーションすること

コミュニケーションは、常に丁寧で明確かつ専門的であるべきです。参加者は、攻撃的または侮辱的な表現を避け、注意深く耳を傾け、誤解を減らすために必要に応じて確認または質問を行うべきです。

意見の対立が生じた場合、議論は個人ではなく技術的な課題やプロジェクトの目標に焦点を当てるべきです。フィードバックは、建設的かつ敬意をもって提供されるべきです。

### 安全で協力的な環境を維持すること

安全なコミュニティ環境を維持するため、すべての参加者の皆さまには、ハラスメント、威圧、脅迫、または議論や協働活動を意図的に妨げる行為を行わないようご配慮をお願いします。 

すべての参加者は、安全で協力的な環境を維持する責任があります。


## 相談および懸念事項への対応

不適切な行動または禁止されている行為が発生した場合、[メーリングリストアドレス] へ報告することが推奨されます。被害の対象でない場合であっても、コミュニティのすべてのメンバーは問題を報告することができます。

すべての報告内容は機密として厳重に取り扱われます。報告された事案は、迅速かつ公平に検討および検証されます。
